# 뇌피셜 캐릭터 이미지 생성 가이드

## 목적

`뇌피셜` Step 1 결과, 공유 카드, 전체 리포트에 사용할 공식 캐릭터 이미지를 제작하기 위한 가이드다.

이 문서는 [`뇌피셜_앱기획서_v6 (1).md`](C:\projects\3-brain-official\뇌피셜_앱기획서_v6%20%281%29.md) 의 `25가지 편향 캐릭터 맵`을 기준으로 작성한다.

중요:
- 이제 더 이상 임시 3캐릭터 기준으로 작업하지 않는다.
- 최종 기준은 아래 25개 공식 캐릭터다.
- 내부 이론은 `25개 편향`이지만, 사용자에게는 `캐릭터`로 보이게 만든다.

## 필요한 산출물

최소 필요:
- 25개 캐릭터 메인 이미지

권장:
- 25개 캐릭터 메인 이미지
- 25개 캐릭터 축약 썸네일 또는 동일 이미지 크롭 버전

## 파일 스펙

- 형식: `PNG`
- 비율: `1:1`
- 권장 크기: `1024x1024` 이상
- 배경: **반드시 투명 배경 (Transparent background)**
- 결과 카드 안에서 바로 보일 수 있게 캐릭터 중심 구도
- 텍스트 삽입 금지
- 시리즈 전체가 같은 세계관처럼 보여야 함

## 공통 스타일 방향

- 전체 앱 톤과 맞는 `soft premium pink-purple` 계열 무드 유지
- **모든 캐릭터의 형태는 반드시 '고양이'로 한정한다. 사람 형태를 절대 사용하지 않는다.**
- 너무 심리테스트스럽거나 싸구려 게임 캐릭터처럼 보이지 않게 조절
- 과하게 실사형으로 가지 말고, 브랜드 자산처럼 정돈된 일러스트/픽셀 아트 스타일 유지
- 캐릭터마다 성격 차이는 분명하되 시리즈 일관성은 유지
- 결과 카드에서 얼굴이나 상징 오브젝트가 한눈에 들어오도록 단순한 중앙 구도 권장
- 배경은 복잡한 장면보다 캐릭터를 살려주는 수준으로 절제

## 피해야 할 것

- **사람(인간) 형태의 캐릭터 (반드시 고양이 아트워크로 생성할 것)**
- 텍스트, 알파벳, 숫자 삽입
- 캐릭터마다 전혀 다른 화풍 사용
- 공포스럽거나 공격적인 표정
- 과도한 네온, 과도한 SF UI, 과도한 3D 렌더 느낌
- 너무 복잡한 배경
- 밈 이미지처럼 가벼운 장난톤으로 흐르는 연출

## 공통 프롬프트 베이스

아래 베이스를 캐릭터별 키워드와 함께 붙여서 사용한다.

```text
Create a square pixel art illustration of a CAT character for a Korean self-reflection app called "뇌피셜".
The character MUST be a cat, not a human.
The character must feel like part of a unified premium 16-bit pixel art series.
Style: clean and crisp pixel art, modern, emotionally readable, slightly playful but not childish, soft premium pink-purple color harmony, simple centered composition, no text, no letters, no numbers.
The image should work well inside a mobile app result card.
Use a strictly transparent background. No background color, scenery, or white background.
The character should feel memorable, human-readable, and brandable.
```

## 캐릭터별 생성 방향

### 1. 본능적 행동가

- 원본 편향: 보상/처벌 초반응
- 핵심 성격: 반응 빠름, 즉시 행동, 본능적 추진력
- 시각 키워드: 점화, 스파크, 스타트 버튼, 돌진 자세
- 프롬프트 보강:

```text
Character: "본능적 행동가".
A fast-reacting action-driven character who responds immediately to reward signals.
Visual mood: energetic, instinctive, quick-starting, bold.
Use dynamic motion cues and a lively pose.
```

### 2. 열정적 몰입가

- 원본 편향: 좋아하는 것 편향
- 핵심 성격: 좋아하면 깊게 파고듦, 몰입, 애정
- 시각 키워드: 집중, 열정, 하트보다 절제된 애정, 몰입하는 눈빛
- 프롬프트 보강:

```text
Character: "열정적 몰입가".
A deeply passionate character who becomes fully immersed in what they love.
Visual mood: warm, focused, devoted, emotionally invested.
```

- 25개가 하나의 시리즈처럼 보이는가

- 캐릭터별 성격 차이가 한눈에 읽히는가
- 결과 카드 축소 썸네일에서도 구분 가능한가
- 텍스트 없이도 캐릭터의 역할이 드러나는가
- 배경이 너무 복잡하지 않은가
- 앱의 pink-purple 톤과 충돌하지 않는가
- 과하게 어둡거나 공포스럽지 않은가
