# 🧠 뇌피셜 — 내 뇌가 공식이라 우기는 것들

## 앱 기획서 v6 (최종)

---

## 1. 개요

- **앱 이름:** 뇌피셜
- **부제:** 내 뇌가 공식이라 우기는 것들
- **타겟 유저:** 10대 후반~30대, 심리테스트/자기분석에 관심 있는 한국 사용자
- **핵심 가치:** AI 대화형 편향 진단 + 고민을 다학제적 렌즈로 찬반 분석
- **철학:** "AI는 답을 주지 않는다. 더 좋은 질문을 준다."

> 참고: 이 앱은 씽크스미스 EP02(찰리 멍거의 멘탈 모델 격자)와 연동되지만, 앱 자체는 독립 프로덕트로 포지셔닝한다. 앱 UI에 "씽크스미스" 브랜딩은 최소화하고, 유튜브 영상에서 자연스럽게 연결한다.

---

## 2. 제품 전략

### 이중 전략: 바이럴 + 리텐션

- **Step 1 (AI 편향 진단)** = 바이럴 훅. AI와 대화하며 내 편향을 발견하고, 결과 카드를 공유
- **Step 2 (고민 찬반 분석)** = 리텐션 엔진. 고민이 생길 때마다 재방문해서 렌즈 분석

### 사용자 여정

- **첫 방문:** Step 1 편향 진단 → 카드 공유 → 신규 유입
- **전환:** Step 2 맛보기 1회 무료 체험 (노로그인) → 가치 경험 후 로그인 유도
- **재방문:** 로그인 → 저장된 편향 프로필 기반으로 Step 2 반복 사용

## 3. 기반 이론

### Step 1 — 인간 오판의 심리학 (The Psychology of Human Misjudgment)

찰리 멍거가 정리한 25가지 인지 편향. "사람이 판단할 때 이렇게 실수한다"는 버그 목록. 각 편향을 긍정적/중립적 캐릭터로 리프레이밍하여 결과를 받았을 때 공유하고 싶은 경험으로 만든다.

### Step 2 — 멘탈 모델 격자 (Mental Model Lattice)

멍거가 투자 판단에 실제 사용한 다학제적 분석 프레임워크. 하나의 문제를 여러 학문의 렌즈로 동시에 분석하되, AI가 해당 고민에 가장 관련 높은 렌즈를 선별하여 제시한다.

---

## 4. 25가지 편향 캐릭터 맵 (Step 1 기반)

1. **본능적 행동가** (보상/처벌 초반응) — 보상에 빠르게 반응하는 실행력의 소유자
2. **열정적 몰입가** (좋아하는 것 편향) — 좋아하면 끝까지 가는 사람
3. **직감형 비평가** (싫어하는 것 편향) — 호불호가 확실한 사람
4. **속도형 결단가** (의심 회피) — 불확실함 속에서도 빠르게 결정하는 사람
5. **신념의 수호자** (비일관성 회피) — 한번 믿으면 흔들리지 않는 사람
6. **끝없는 탐구자** (호기심 경향) — 궁금하면 끝까지 파고드는 사람
7. **정의의 저울** (칸트식 공정 경향) — 공정함에 대한 감각이 남다른 사람
8. **열정적 경쟁자** (질투/시기심) — 주변의 성과가 자극이 되는 사람
9. **의리의 사람** (상호성 경향) — 받은 만큼 돌려주는 사람
10. **분위기 리더** (영향력 편향) — 주변 에너지를 빠르게 읽는 사람
11. **신중한 기억가** (부정의 힘) — 실패 경험에서 깊이 배우는 사람
12. **타고난 낙관주의자** (과잉 낙관) — 어떤 상황에서도 가능성을 보는 사람
13. **평화로운 전략가** (고통 회피) — 불필요한 충돌을 피하는 지혜가 있는 사람
14. **자기확신의 리더** (과신 편향) — 스스로의 판단을 신뢰하는 사람
15. **끈기의 투자자** (매몰비용) — 쉽게 포기하지 않는 사람
16. **트렌드 감지자** (가용성 편향) — 최근 정보에 빠르게 반응하는 사람
17. **직관의 프로파일러** (대표성 편향) — 첫인상에서 많은 걸 읽어내는 사람
18. **지식의 수집가** (권위 복종) — 전문가의 지혜를 존중하는 사람
19. **공감형 트렌드세터** (사회적 증거) — 사회적 흐름을 잘 읽는 사람
20. **맥락의 해석가** (대비 효과) — 상황과 맥락에 민감한 분석가
21. **감성 스프린터** (스트레스 편향) — 압박 속에서 폭발적 에너지를 내는 사람
22. **안정 추구형 전략가** (현상유지 편향) — 검증된 것을 선호하는 사람
23. **순간 집중형 실행가** (약물 남용 편향) — 현재의 경험에 강렬하게 몰입하는 사람
24. **경험의 수호자** (노화 편향) — 과거의 지혜를 소중히 여기는 사람
25. **올인 돌파형** (롤라팔루자 효과) — 여러 확신이 겹치면 과감하게 밀어붙이는 사람

> 각 캐릭터별 상세 설명 (발동 상황, 대표 인물, 멍거 명언 포함)은 별도 문서로 작업 예정

## 5. 6개 분석 렌즈 (Step 2 기반)

1. **🧠 심리학** — 인간 행동의 비합리성, 인센티브 구조, 감정적 반응
2. **💰 경제학** — 기회비용, 매몰비용, 규모의 경제, 인센티브
3. **🎲 수학/확률** — 기대값, 복리, 확률적 사고, 베이즈 추론
4. **🧬 생물학/진화** — 생존 본능, 적응, 경쟁 구조, 자연선택
5. **⚙️ 물리학/공학** — 임계점, 피드백 루프, 안전마진, 시스템 사고
6. **📜 역사학** — 과거 패턴의 반복, 유추적 사고, 선례 분석

> Step 2에서 6개 렌즈를 모두 보여주지 않는다. AI가 사용자의 고민을 분석하여 가장 관련 높은 핵심 3~4개 렌즈를 선별하여 제시한다. 나머지 렌즈는 "더 보기"로 접어둔다.

## 6. 사용자 플로우

### 첫 방문 (비로그인)

```
랜딩 페이지

```

## 7. Step 1 상세: AI 대화형 편향 진단

### 7-1. 방식

**AI 적응형 대화 (A+C 하이브리드)**

- 터치(선택지) 90% + 짧은 타이핑 10%
- AI가 사용자 답변을 보고 다음 질문 + 선택지를 실시간 생성
- 매번 다른 질문, 다른 경험
- 최소 7턴 ~ 최대 12턴 (사용자마다 다름)

### 7-2. 대화 예시

> 🤖 "요즘 뭔가 사고 싶은 거 있어요?"
> → [전자기기] [옷/패션] [음식/맛집] [없음] ← 터치
>
> 🤖 "뭘 사고 싶어요? 간단하게!"
> → "아이패드" ← 짧은 타이핑 (단어 수준)
>
> 🤖 "아이패드 사고 싶은 이유는?"
> → [다들 좋다고 해서] [업무에 필요해서] [갖고 싶으니까] [할인 중이라서] ← 터치
>
> 🤖 "근데 아직 안 산 이유는?"
> → [비싸서] [이미 노트북 있어서] [살 필요 없을 것 같아서] [고민 중] ← 터치

### 7-3. 내부 로직

- AI가 매 턴마다 25개 편향 점수를 업데이트
- 1턴에서 3~5개 편향 동시 측정
- 다음 질문은 아직 불확실한 편향을 집중 탐색하도록 설계
- Top 3 확신도가 충분해지면 진단 완료

### 7-4. UX 최적화: 로딩 지연 대응

AI 적응형 방식은 매 턴 API 호출이 발생하므로, 사용자가 로딩을 자연스럽게 받아들이도록 다음 전략을 적용한다.

**① 마이크로 애니메이션 (의미 있는 로딩)**
로딩 중 "🧠 뇌피셜 분석 중...", "찰리 멍거의 뇌 구조와 동기화 중..." 등의 분석 애니메이션을 표시하여 대기 시간을 콘텐츠 경험으로 전환한다. 단순 스피너가 아닌 "분석하고 있다"는 느낌을 줘서 오히려 신뢰감을 높인다.

**② 프롬프트 최적화 (응답 속도 극대화)**
AI에게 보내는 지시문을 최소화하고, 출력을 JSON 형식으로 고정하여 응답 속도를 극대화한다.

- AS-IS: 장문 지시 + 자유 형식 응답 → 느림
- TO-BE: 최소 지시 + JSON 출력 고정 → 빠름

예시:
```
입력: "이전답변:B, 점수:{매몰:72,사회적증거:65...}, 다음질문+선택지4개 JSON반환"
출력: {"question":"...","options":["A","B","C","D"],"bias_map":{...}}
```

Gemini 2.5 Flash-Lite 기준 첫 토큰 ~0.33초 + 생성 ~1초 = 총 약 1.3초. 스트리밍 적용 시 체감 로딩은 1초 이내. 마이크로 애니메이션과 결합하면 사용자가 "분석 중" 경험으로 자연스럽게 받아들인다.

**③ 타이핑 구간 UX 강화**
전체 진단 중 1~2회 등장하는 짧은 타이핑 구간의 마찰을 최소화한다.

- Auto-focus: 입력창 등장 시 자동으로 키보드가 올라옴
- 플레이스홀더 예시: "예: 아이패드", "예: 퇴사" 등 뇌를 거치지 않고 바로 칠 수 있는 힌트 제공

### 7-5. 진행 표시

문항 수가 아닌 진행 바로 표시한다.

```
🧠 뇌피셜 분석 중... ████████░░ 78%
```

편향이 뚜렷한 사람은 7~8턴에 100% 도달, 고르게 분포된 사람은 10~12턴까지 진행.

### 7-6. 랜딩 페이지

- 헤드라인: "당신의 뇌는 매일 당신을 속이고 있습니다"
- 서브: "찰리 멍거가 정리한 25가지 편향 중, 당신의 Top 3는?"
- CTA: "내 뇌피셜 알아보기"
- 소요시간: 약 3분

### 7-7. 결과 페이지

- 나의 편향 Top 3 (캐릭터명 + 부제 + 한 줄 설명)
- 25개 편향 레이더 차트
- 편향별 상세 해설 (접기/펼치기, MBTI 스타일)
- 멍거 명언 1개
- 공유 카드 다운로드 버튼
- Step 2 맛보기 유도: "이 편향이 실제 고민에서 어떻게 작동하는지 볼래요?"

## 8. Step 2 상세: 고민 찬반 분석

### 8-1. 진입 경로

- **첫 체험 (노로그인):** Step 1 결과 페이지에서 1회 무료 체험 제공
- **이후 사용 (로그인 필요):** 구글 로그인 후 대시보드에서 반복 사용

### 8-2. 분석 플로우

**1단계: 고민 입력**

자유 텍스트로 고민 입력.
> 예: "지금 다니는 회사 때려치고 프리랜서로 전환할까?"

**2단계: AI가 핵심 렌즈 선별 + 찬반 분석**

AI가 고민 내용을 분석하여 6개 렌즈 중 가장 관련 높은 3~4개를 선별하고, 각 렌즈별로 찬성/반대/조건부 판정 + 근거 한 줄 요약.

> "이 고민은 다음 관점에서 보는 게 핵심입니다:"
>
> 🧠 심리학 → ⚠️ 조건부: 현 직장 스트레스가 판단을 왜곡하고 있을 수 있음
> 💰 경제학 → ❌ 반대: 안정적 수입 포기의 기회비용이 큼
> ⚙️ 물리학/공학 → ❌ 반대: 안전마진 없이 전환하면 복구 불가능
> 📜 역사학 → ✅ 찬성: 대부분의 커리어 전환은 장기적으로 긍정적 결과
>
> [나머지 렌즈 더 보기 ▼]

**3단계: 편향 경고 (Step 1 연동)**

사용자의 Top 3 편향이 이 고민에서 어떻게 작동하는지 경고.

> ⚠️ 편향 경고
> 당신의 1위 편향 "끈기의 투자자"가 이 판단에 개입하고 있을 수 있습니다.
> → 경제학 렌즈에서 "기회비용"을 과소평가하고, "지금까지 쌓은 커리어"에 집착하는 방향으로 판단이 기울 수 있습니다.

**4단계: 결정적 질문 도출**

AI가 찬반 충돌 지점에서 결정적 질문 2~3개를 도출. 답은 제공하지 않음.

> 💡 당신의 판단은 결국 이 질문에 달려있습니다:
> 1. 프리랜서 수입이 현재의 60%로 떨어져도 6개월 버틸 수 있는가?
> 2. 1년 후 복귀하고 싶을 때, 돌아갈 경로가 있는가?
> 3. 떠나고 싶은 이유가 "이 직장이 싫어서"인가, "프리랜서가 좋아서"인가?

### 8-3. 핵심 원칙

**AI는 답을 주지 않는다. 더 좋은 질문을 준다.**

GPT가 "이런저런 걸 고려하면 이직을 추천합니다" 식으로 답을 주는 것과 달리, 뇌피셜은 "당신이 놓치고 있는 관점과 질문"을 던져주고, 결론은 사용자 몫으로 남겨둔다.

## 9. 공유 기능

### 9-1. Step 1 공유 (바이럴용)

- 결과 카드 이미지 다운로드
- 카카오톡 공유 (카카오 공유 API)
- 링크 복사

카드 내용: Top 3 편향 캐릭터 + 레이더 차트 + 멍거 명언 + "나도 테스트하기" 링크

### 9-2. Step 2 공유 (실용 공유)

- 분석 결과 카드 이미지 다운로드
- 카카오톡 공유
- 링크 복사

용도: 신뢰하는 사람(배우자, 멘토, 친구)에게 보내서 함께 고민

### 9-3. 바이럴 루프

- Step 1 카드 = "나 끈기의 투자자래 ㅋㅋ" (재미 공유)
- Step 2 카드 = "이거 봐봐 나 진짜 고민인데" (진지한 공유)
- 두 가지 공유 채널로 유입 경로 다변화

## 10. 로그인 및 데이터 저장

### 10-1. 로그인 전략

로그인 없이도 Step 1 + Step 2 맛보기 1회를 완전히 경험 가능.
가치를 체감한 뒤 로그인을 유도하여 전환율을 극대화한다.

- Step 1 (편향 진단): 로그인 불필요
- Step 2 첫 1회 (맛보기): 로그인 불필요
- Step 2 2회 이상 + 히스토리 저장: 구글 로그인 필요

### 10-2. 저장 데이터

**유저 프로필 (구글 로그인):**

- 분석 일시

### 10-3. 대시보드 (로그인 후 메인)

- "편향 재검사" 버튼

## 11. GPT 대비 차별화

- **진단 방식:** GPT는 일반론 / 뇌피셜은 AI 적응형 대화로 행동 기반 개인화 진단
- **진단 경험:** GPT는 매번 같은 대화 / 뇌피셜은 매번 다른 질문, 다른 경험
- **분석 구조:** GPT는 장단점 나열 / 뇌피셜은 핵심 렌즈 선별 + 찬반 충돌 시각화
- **개인화:** GPT는 매번 제로에서 시작 / 뇌피셜은 저장된 편향 프로필이 분석에 반영
- **결과물:** GPT는 텍스트 / 뇌피셜은 카드 + 차트 + 공유
- **철학:** GPT는 답을 줌 / 뇌피셜은 더 좋은 질문을 줌

## 13. AI 프롬프트 설계

### 13-1. Step 1: 적응형 편향 진단 프롬프트

**시스템 프롬프트:**

```
너는 "뇌피셜" 편향 진단 AI야.
사용자와 자연스러운 대화를 하면서 찰리 멍거의 25가지 인지 편향 중
사용자의 Top 3를 찾아내는 것이 목표야.

[규칙]
1. 매 턴 일상적이고 공감 가능한 시나리오 질문을 1개 생성해
2. 선택지는 반드시 4개, 각각 서로 다른 편향에 매핑해
3. 질문 톤은 친구한테 말하듯 가볍고 재밌게 (반말, 이모지 적절히)
4. 이전 답변을 기반으로 아직 불확실한 편향을 집중 탐색해
5. 선택지 중 90%는 터치형(짧은 텍스트), 필요시 1개만 짧은 타이핑 유도
6. 절대 학술 용어(인지 편향, 매몰비용, 확증편향 등)를 쓰지 마
7. 하나의 선택지가 여러 편향에 동시에 점수를 줄 수 있어 (1턴 다중 측정)
8. 이전 턴과 비슷한 상황/질문을 반복하지 마
9. 첫 2~3턴은 가벼운 일상 질문으로 시작하고, 점차 판단/선택 상황으로 깊어져

[25가지 편향 목록]
1. 보상/처벌 초반응 (본능적 행동가)
2. 좋아하는 것 편향 (열정적 몰입가)
3. 싫어하는 것 편향 (직감형 비평가)
4. 의심 회피 (속도형 결단가)
5. 비일관성 회피 (신념의 수호자)
6. 호기심 경향 (끝없는 탐구자)
7. 칸트식 공정 경향 (정의의 저울)
8. 질투/시기심 (열정적 경쟁자)
9. 상호성 경향 (의리의 사람)
10. 영향력 편향 (분위기 리더)
11. 부정의 힘 (신중한 기억가)
12. 과잉 낙관 (타고난 낙관주의자)
13. 고통 회피 (평화로운 전략가)
14. 과신 편향 (자기확신의 리더)
15. 매몰비용 (끈기의 투자자)
16. 가용성 편향 (트렌드 감지자)
17. 대표성 편향 (직관의 프로파일러)
18. 권위 복종 (지식의 수집가)
19. 사회적 증거 (공감형 트렌드세터)
20. 대비 효과 (맥락의 해석가)
21. 스트레스 편향 (감성 스프린터)
22. 현상유지 편향 (안정 추구형 전략가)
23. 약물 남용 편향 (순간 집중형 실행가)
24. 노화 편향 (경험의 수호자)
25. 롤라팔루자 효과 (올인 돌파형)

[현재 상태 — 매 턴 백엔드에서 주입]
- 턴 수: {turn_number}
- 누적 점수: {bias_scores}
- 확신도 상위 3개: {top3_confidence}
- 아직 불확실한 편향: {uncertain_biases}
- 이전 대화: {conversation_history}

[출력 형식 — 반드시 JSON만 출력, 다른 텍스트 없이]
{
  "question": "질문 텍스트",
  "options": [
    {"text": "선택지A", "type": "touch", "bias_map": {"15": 8, "22": 5}},
    {"text": "선택지B", "type": "touch", "bias_map": {"19": 9, "10": 4}},
    {"text": "선택지C", "type": "touch", "bias_map": {"12": 7, "14": 6}},
    {"text": "선택지D", "type": "touch", "bias_map": {"5": 8, "18": 3}}
  ],
  "analysis": {
    "updated_scores": {"1": 12, "2": 45, ...},
    "top3": [15, 19, 12],
    "top3_confidence": [0.85, 0.78, 0.72],
    "uncertain": [3, 7, 21],
    "should_end": false,
    "progress": 65
  }
}
```

**핵심 로직 설명:**

- `should_end` = true 조건: Top 3의 확신도가 전부 0.80 이상이거나, 12턴 도달 시
- `progress`: 확신도 평균 기반으로 계산 → 프론트 진행 바에 반영
- `bias_map`: 하나의 선택지가 여러 편향에 동시에 점수 부여 (1턴 다중 측정의 핵심)
- `uncertain_biases`: AI에게 "다음 턴에서 이 편향들을 집중 탐색하라"는 가이드
- `type`: "touch"(선택지 터치) 또는 "input"(짧은 타이핑) — input은 전체 진단 중 최대 1~2회만

### 13-2. Step 1: 결과 생성 프롬프트

진단 완료(should_end = true) 후, 별도 프롬프트로 최종 결과를 생성한다.

```
사용자의 편향 진단이 완료됐어. 아래 데이터로 결과 리포트를 생성해.

[최종 점수]: {final_scores}
[Top 3]: {top3_biases}
[대화 로그]: {conversation_log}

[규칙]
1. 각 Top 3 편향의 상세 설명에 대화에서 나온 구체적 사례를 반영해
   (예: "아이패드를 사고 싶으면서도 노트북이 있다는 이유로 미루고 있었죠")
2. 톤은 따뜻하고 공감적, 단점 지적이 아닌 특성 해설
3. trigger_situations는 한국 20~30대가 공감할 일상 상황으로
4. 한국어 존댓말

[출력 형식 — JSON]
{
  "top3": [
    {
      "rank": 1,
      "bias_id": 15,
      "character_name": "끈기의 투자자",
      "subtitle": "쉽게 포기하지 않는 사람",
      "summary": "한 줄 요약",
      "detail": "2~3문단 상세 설명 (대화 사례 반영 필수)",
      "trigger_situations": ["상황1", "상황2", "상황3", "상황4"],
      "famous_person": "인물명 — 설명",
      "munger_quote": "관련 멍거 명언"
    },
    ...
  ],
  "radar_scores": {"1": 12, "2": 45, ...},
  "overall_insight": "종합 인사이트 2~3문장"
}
```

**핵심 포인트:** `detail`에 대화 로그 기반 구체적 사례를 반영하는 것이 "이건 진짜 나를 분석한 거네" 임팩트의 핵심이다. 고정 질문 심리테스트와의 결정적 차별화 지점.

### 13-3. Step 2: 고민 찬반 분석 프롬프트

```
너는 찰리 멍거의 멘탈 모델 격자 기반 고민 분석 AI야.
사용자의 고민을 다학제적 렌즈로 분석하되,
절대 결론이나 추천을 하지 않고, 결정적 질문만 도출해.

[6개 렌즈]
1. 🧠 심리학 — 인간 행동의 비합리성, 인센티브, 감정
2. 💰 경제학 — 기회비용, 매몰비용, 규모의 경제
3. 🎲 수학/확률 — 기대값, 복리, 확률적 사고
4. 🧬 생물학/진화 — 생존 본능, 적응, 경쟁
5. ⚙️ 물리학/공학 — 임계점, 피드백 루프, 안전마진
6. 📜 역사학 — 과거 패턴, 유추, 선례

[사용자 편향 프로필]
- Top 1: {bias_1_name} (점수: {score})
- Top 2: {bias_2_name} (점수: {score})
- Top 3: {bias_3_name} (점수: {score})

[사용자 고민]
"{user_input}"

[규칙]
1. 6개 렌즈 중 이 고민에 가장 관련 높은 3~4개만 선별해
2. 선별한 렌즈별로 찬성/반대/조건부 판정 + 근거 한 줄
3. 사용자의 Top 3 편향이 이 고민에서 어떻게 작동할 수 있는지 경고
4. 찬반 충돌 지점에서 결정적 질문 2~3개 도출
5. 결정적 질문은 구체적이고 답변 가능한 형태여야 함
   (Bad: "충분히 준비되었는가?" → Good: "프리랜서 수입이 60%로 줄어도 6개월 버틸 수 있는가?")
6. 절대 "~하세요", "추천합니다" 같은 결론을 내리지 마
7. 한국어 존댓말, 공감적 톤
8. "ㅋㅋ 그냥 해봄" 같은 성의 없는 입력에는 부드럽게 구체적으로 적어달라고 요청

[출력 형식 — JSON]
{
  "selected_lenses": [
    {
      "lens": "심리학",
      "emoji": "🧠",
      "verdict": "조건부",
      "reasoning": "현 직장 스트레스가 판단을 왜곡하고 있을 수 있습니다"
    },
    ...
  ],
  "excluded_lenses": [
    {
      "lens": "생물학/진화",
      "emoji": "🧬",
      "reason": "이 고민과 직접적 관련도가 낮아 제외했습니다"
    },
    ...
  ],
  "bias_warning": {
    "primary_bias": "끈기의 투자자",
    "how_it_works": "이미 쌓은 커리어에 대한 집착이 기회비용 판단을 흐릴 수 있습니다",
    "affected_lens": "경제학",
```

### 13-4. 프롬프트 테스트 계획

프롬프트 품질이 앱의 성패를 결정한다. 다음 절차로 반복 테스트한다.

**1차 (3~5명 시뮬레이션)**
다양한 페르소나(10대 여학생, 30대 직장인, 20대 대학생 등)로 Step 1 대화를 시뮬레이션한다. Gemini 2.5 Flash-Lite의 JSON 출력 안정성과 한국어 질문 품질을 중점 확인.

**2차 (문제 패턴 수집 + 프롬프트 보강)**
1차에서 발견된 문제 패턴을 수집하여 프롬프트에 "하지 마" 규칙을 추가한다. 주요 예상 문제: 특정 편향만 반복 탐색, 선택지가 너무 뻔함, JSON 형식 깨짐, 톤이 갑자기 딱딱해짐.

**3차 (10명 시뮬레이션)**
보강된 프롬프트로 더 많은 케이스를 테스트한다. Step 2도 Gemini 2.5 Flash로 다양한 고민 유형(이직, 연애, 구매, 투자, 학업 등)을 테스트하여 렌즈 선별과 결정적 질문의 품질을 확인한다.

**4차 (엣지 케이스 테스트)**
성의 없는 입력("ㅋㅋ", "몰라", "그냥"), 부적절한 입력, 극단적 편향 분포 등 예외 상황을 집중 테스트하여 방어 로직을 완성한다.

이 과정을 일정 중 3~4일에 걸쳐 진행한다.

### 15-2. 비용 방어 기제

바이럴이 터졌을 때 비용 폭발을 방지하기 위한 안전장치.

**사용 횟수 제한:**
- Step 1 (편향 진단): 1인당 하루 1회 (재검사는 1주일 간격 권장)
- Step 2 (고민 분석): 비로그인 1회 무료 / 로그인 후 하루 3회 무료
- 초과 시: "오늘 분석을 모두 사용했어요. 내일 다시 만나요!" 안내

**비용 회수 옵션 (선택적):**
- 토스 익명 후원 (커피 한 잔 사주기) — 결과 페이지 하단에 작게 배치
- 추후 프리미엄 플랜 검토 가능 (무제한 분석, 상세 리포트 등)

**긴급 대응:**
- 일간 API 호출 상한선 설정 (예: 하루 10만 회)
- 상한 초과 시 자동 대기열 처리 또는 일시 중단
- 실시간 비용 모니터링 대시보드 (관리자용)

- Step 2 맛보기: 노로그인 1회 무료 체험
- Step 2: 고민 입력 → 핵심 3~4개 렌즈 선별 + 찬반 분석 → 편향 경고 → 결정적 질문 도출
- Step 2 공유: 분석 결과 카드 이미지 다운로드 + 카카오톡 공유 + 링크 복사

- 대시보드: 편향 프로필 + 분석 히스토리
- 비용 방어: 사용 횟수 제한 + 일간 API 상한선
