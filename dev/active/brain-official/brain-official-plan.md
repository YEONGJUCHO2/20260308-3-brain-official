# 뇌피셜 (Brain-fficial) — 종합 개발 계획

> Last Updated: 2026-03-07

## Executive Summary

"뇌피셜"은 찰리 멍거의 25가지 인지 편향과 멘탈 모델 격자를 기반으로 한 AI 대화형 웹앱이다.
두 가지 핵심 기능을 제공한다:
1. **Step 1 (바이럴 훅):** AI 적응형 대화로 사용자의 인지 편향 Top 3를 진단
2. **Step 2 (리텐션 엔진):** 사용자의 고민을 다학제적 렌즈로 찬반 분석

**철학:** "AI는 답을 주지 않는다. 더 좋은 질문을 준다."

## Risk Assessment

| 리스크 | 확률 | 영향 | 대응 |
|--------|------|------|------|
| Gemini JSON 출력 불안정 | 중 | 높음 | JSON 스키마 강제 + 파싱 fallback + 재시도 |
| API 응답 지연 (>3초) | 중 | 중 | 스트리밍 + 마이크로 애니메이션 |
| 바이럴 시 비용 폭발 | 낮 | 높음 | 사용 횟수 제한 + 일간 상한선 |
| 카카오 공유 API 제한 | 낮 | 중 | 이미지 다운로드 대안 |
| 편향 진단 품질 불균일 | 중 | 높음 | 4단계 프롬프트 테스트 절차 |
| Gemini Flash-Lite 한국어 품질 | 중 | 높음 | Flash로 fallback 옵션 준비 |
